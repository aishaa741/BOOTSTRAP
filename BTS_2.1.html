Risposte

1. Qual è la dimensione del testo di default di bootstrap?

La dimensione del testo di default in Bootstrap è **16px**.

2. Qual è il font di default di bootstrap?

Il font di default in Bootstrap è **"Helvetica Neue"**, con alternative di riserva come **Helvetica**, **Arial**, e **sans-serif**.

3. Quali sono il margine superiore e inferiore di un paragrafo?

In Bootstrap, i paragrafi (`

`) hanno un margine superiore e inferiore di **1rem** ciascuno per impostazione predefinita. Questo equivale a circa 16px, ma può variare in base alla dimensione del font del contenitore.

Cose interessanti

HTML
HTML (HyperText Markup Language) è un **linguaggio di markup**, non un linguaggio di programmazione. Serve a strutturare e organizzare i contenuti di una pagina web, indicando elementi come titoli, paragrafi, immagini, link e altri componenti. HTML definisce la struttura della pagina, che i browser interpretano per visualizzare correttamente il contenuto sullo schermo..
CSS
CSS (Cascading Style Sheets) è un **linguaggio di stile** utilizzato per descrivere l'aspetto e il layout di una pagina web..
JS
JavaScript (JS) è un **linguaggio di programmazione** utilizzato per creare interattività nelle pagine web..

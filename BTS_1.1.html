Risposte in un Container Fluid

1. A cosa serve bootstrap?

Bootstrap è una raccolta di strumenti per creare siti web in modo più facile e veloce. Fornisce stili e componenti già pronti (come pulsanti, menu, griglie di layout) che si adattano automaticamente a diversi schermi e dispositivi, così non devi creare tutto da zero..

2. Cosa devi importare per creare una pagina web con bootstrap?

Per creare una pagina web con bootstrap devi importare:
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

3. Cos'è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Un CDN, o Content Delivery Network, è una rete di server distribuiti in diverse località che servono contenuti web, come file CSS e JavaScript, ai visitatori del sito.
Vantaggi di usare un CDN per Bootstrap rispetto a scaricare i file sul tuo sito:
Velocità: I file vengono serviti dal server più vicino all'utente, riducendo il tempo di caricamento.
Affidabilità: I CDN hanno molte copie dei file, quindi se uno è fuori servizio, gli altri possono continuare a funzionare.
Scalabilità: In caso di picchi di traffico, i CDN possono gestire più richieste senza sovraccaricare il tuo server.
Aggiornamenti automatici: Se usi un CDN, puoi ottenere automaticamente le ultime versioni di Bootstrap senza dover aggiornare manualmente i file sul tuo server.

4. A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1

L'attributo `` nel tag `` serve per fare in modo che la pagina web sia visibile correttamente su tutti i dispositivi, come smartphone e tablet. - **`width=device-width`** imposta la larghezza della pagina per adattarsi alla larghezza dello schermo del dispositivo. - **`initial-scale=1`** assicura che la pagina non sia né ingrandita né rimpicciolita all’apertura. Questo attributo rende il sito più "responsive", cioè facilmente leggibile su dispositivi di diverse dimensioni.

5. Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

In Bootstrap, la differenza tra container e container-fluid è:
· container: Crea un contenitore con larghezza fissa che varia in base alle dimensioni dello schermo, centrato con margini laterali automatici.
· container-fluid: Crea un contenitore a larghezza completa, che si estende su tutta la larghezza della finestra, senza margini laterali.
